## Getting Started
go to the service directory
```bash
cd en
```


### Prerequisites
1. Python 3.7+

### Installing
setup virtual environment
```bash
virtualenv venv
```
install dependencies
```bash
pip install -r requirements.txt
```

### Run Locally

```bash
export FLASK_APP=app
flask run
```

enjoy!
